# sax-design-vue

Vue 3 component library based on the Vuesax 3.x design language.

## Install

```bash
pnpm add sax-design-vue
```

## Usage

```ts
import { createApp } from 'vue'
import SaxDesignVue from 'sax-design-vue'
import 'sax-design-vue/theme-chalk/index.css'

createApp(App).use(SaxDesignVue).mount('#app')
```

Documentation: https://adoin.github.io/sax-design-vue/
