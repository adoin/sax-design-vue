# 致敬 Vuesax

## 设计渊源

**Sax Design Vue** 是 Vuesax 生态的 Vue 3 + TypeScript 延续。原版 [Vuesax](https://github.com/lusaxweb/vuesax) 由 **Lusaxweb** 打造，为 Vue 应用引入了大胆、富有表现力的视觉语言 — 拒绝与千篇一律的 Material UI 雷同。

我们深深感谢这套设计传承：

- **鲜明美学** — 圆润表面、鲜明强调色，组件生动而非平庸
- **创作自由** — Vuesax 从不拘泥于单一设计体系；这种精神指导着今日的组件样式
- **社区影响** — 多年示例、讨论与设计探索，帮助无数开发者更快交付精美界面

本站现代化了开发体验（Vue 3、Vite、TypeScript、暗色模式、Playground），同时尊重自 Vuesax 继承的视觉基因。
