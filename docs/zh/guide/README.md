## 什么是 Sax Design Vue？

Sax Design Vue 是从 Vuesax 生态迁移而来的 **Vue 3 + TypeScript** 组件库。面向现代工具链（Vite、pnpm workspaces），聚焦开发者最需要的三件事：

1. **如何使用**组件构建真实项目
2. **如何配置**属性、插槽与主题
3. **如何预览**行为 — 使用[在线 Playground](/zh/guide/playground)

</card>

## 设计理念

与原版 Vuesax 一样，我们避免千篇一律的 Material 风格。组件在视觉上保持表现力，同时通过属性与 CSS 变量（含**浅色 / 深色**主题）易于配置。
