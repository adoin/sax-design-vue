# Tribute to Vuesax

<card>

## Design lineage

**Sax Design Vue** is a Vue 3 + TypeScript continuation of the Vuesax ecosystem. The original [Vuesax](https://github.com/lusaxweb/vuesax) project by **Lusaxweb** introduced a bold, expressive visual language for Vue applications — one that refused to look like every other Material-based UI kit.

We are deeply grateful for that design heritage:

- **Distinctive aesthetics** — rounded surfaces, vivid accents, and components that feel alive rather than generic.
- **Creative freedom** — Vuesax was never locked to a single design system rulebook; that spirit guides our component styling today.
- **Community impact** — years of examples, discussions, and design exploration that helped countless developers ship beautiful interfaces faster.

This documentation site modernizes the developer experience (Vue 3, Vite, TypeScript, dark mode, playground), while respecting the visual DNA inherited from Vuesax.
